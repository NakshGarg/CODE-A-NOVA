💰 Expense Tracker (Python)
📌 Overview

An Expense Tracker is a Python-based console application used to record, categorize, and manage daily expenses.
It uses core Python concepts such as file handling, functions, and data structures to store and analyze financial data.

🛠️ Technologies Used

Python

File Handling

Lists & Dictionaries

⚙️ Features

Add expenses

View all expenses

Calculate total expenses

Category-wise expense summary

Permanent data storage

📂 Project Files
expense_tracker.py
expenses.txt
README.md

▶️ How to Run

Open terminal / command prompt

Run:

python expense_tracker.py

🧾 Data Format
Date,Category,Amount,Description


Example:

05-01-2026,Food,180,Lunch at hostel mess

🧠 Learning Outcome

Improves understanding of real-world financial tracking and core Python programming.
